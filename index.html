Решение на QGIS:

1. Откройте QGIS и добавьте в проект слои deforestation.shp, LC08_L1TP_182015_20180710_20200831_02_T1_B4_clip.tif, LC08_L1TP_182015_20180710_20200831_02_T1_B5_clip.tif, LC08_L1TP_182015_20250713_20250713_02_RT_B4_clip.tif, LC08_L1TP_182015_20250713_20250713_02_RT_B5_clip.tif
2. Используйте инструмент Raster Calculator для расчета NDVI для обоих снимков:
- NDVI 2018: (LC08_L1TP_182015_20180710_20200831_02_T1_B5_clip - LC08_L1TP_182015_20180710_20200831_02_T1_B4_clip) / (LC08_L1TP_182015_20180710_20200831_02_T1_B5_clip + LC08_L1TP_182015_20180710_20200831_02_T1_B4_clip)
- NDVI 2025: (LC08_L1TP_182015_20250713_20250713_02_RT_B5_clip - LC08_L1TP_182015_20250713_20250713_02_RT_B4_clip) / (LC08_L1TP_182015_20250713_20250713_02_RT_B5_clip + LC08_L1TP_182015_20250713_20250713_02_RT_B4_clip)
3. Сохраните результаты расчетов NDVI в отдельные растровые слои.
4. Используйте инструмент Zonal Statistics для расчета средних значений NDVI в границах вырубок (слой deforestation.shp).
5. Добавьте новые поля в атрибутную таблицу слоя deforestation.shp для хранения средних значений NDVI для 2018 и 2025 годов.
6. Вычислите разницу между NDVI 2025 и NDVI 2018 для каждой вырубки.
7. Создайте выборку вырубок, где значение NDVI 2025 больше NDVI 2018.
8. Подсчитайте количество таких вырубок.

Код на Python для выполнения задачи:

import os
from qgis.core import *
from qgis.analysis import QgsZonalStatistics

# Путь к файлам
path = '/path/to/files'
shp_path = os.path.join(path, 'deforestation.shp')
b4_2018 = os.path.join(path, 'LC08_L1TP_182015_20180710_20200831_02_T1_B4_clip.tif')
b5_2018 = os.path.join(path, 'LC08_L1TP_182015_20180710_20200831_02_T1_B5_clip.tif')
b4_2025 = os.path.join(path, 'LC08_L1TP_182015_20250713_20250713_02_RT_B4_clip.tif')
b5_2025 = os.path.join(path, 'LC08_L1TP_182015_20250713_20250713_02_RT_B5_clip.tif')

# Расчет NDVI
ndvi_2018 = os.path.join(path, 'ndvi_2018.tif')
ndvi_2025 = os.path.join(path, 'ndvi_2025.tif')

processing.run('gdal:rastercalculator', {'INPUT_A': b5_2018, 'BAND_A': 1, 'INPUT_B': b4_2018, 'BAND_B': 1, 'FORMULA': '(A-B)/(A+B)', 'OUTPUT': ndvi_2018})
processing.run('gdal:rastercalculator', {'INPUT_A': b5_2025, 'BAND_A': 1, 'INPUT_B': b4_2025, 'BAND_B': 1, 'FORMULA': '(A-B)/(A+B)', 'OUTPUT': ndvi_2025})

# Загрузка слоев
deforestation_layer = QgsVectorLayer(shp_path, 'Deforestation', 'ogr')
ndvi_2018_layer = QgsRasterLayer(ndvi_2018, 'NDVI 2018')
ndvi_2025_layer = QgsRasterLayer(ndvi_2025, 'NDVI 2025')

# Расчет зональной статистики
zonal_stats_2018 = QgsZonalStatistics(deforestation_layer, ndvi_2018_layer, 'ndvi_2018_', 1, QgsZonalStatistics.Mean)
zonal_stats_2018.calculateStatistics(None)

zonal_stats_2025 = QgsZonalStatistics(deforestation_layer, ndvi_2025_layer, 'ndvi_2025_', 1, QgsZonalStatistics.Mean)
zonal_stats_2025.calculateStatistics(None)

# Подсчет вырубок с увеличением биомассы
features = deforestation_layer.getFeatures()
count = 0

for feature in features:
    ndvi_2018_mean = feature['ndvi_2018_mean']
    ndvi_2025_mean = feature['ndvi_2025_mean']
    
    if ndvi_2025_mean > ndvi_2018_mean:
        count += 1

print(count)
